# Desafio POO em Java — Plataforma de Streaming

## Descrição

Projeto de prática de **Programação Orientada a Objetos (POO)** em Java.  
Simula uma plataforma de streaming com:

- Filmes  
- Séries e episódios  
- Atores/Atrizes  
- Diretores  
- Histórico de participação dos profissionais  

O objetivo é aplicar **abstração, encapsulamento, herança e polimorfismo**.

---

## Estrutura do Projeto

Principais classes:

- `Conteudo` (abstrata)  
- `Filme`  
- `Serie`  
- `Episodio`  
- `Profissional` (abstrata)  
- `Ator_Atriz`  
- `Diretor_a`  
- `App` (classe principal)

---

## Funcionalidades

- Criar filmes e séries  
- Adicionar atores e diretores  
- Registrar episódios de séries  
- Manter histórico de filmes e séries por profissional  
- Listar conteúdos usando **Streams**  
- Iniciar e pausar reprodução de filmes ou episódios  

A duração dos filmes é representada com `java.time.Duration`.
